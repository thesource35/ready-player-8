| Boundary | Description |
|----------|-------------|
| UserDefaults / @AppStorage | Device-local state; no cross-device sync; blast radius is single user, single device |
| UIApplication.openSettingsURLString | System-provided URL; navigates to the app's own Settings pane — cannot be tampered |
| CLLocationManagerDelegate → view state | Delegate callbacks run on main thread (confirmed by @MainActor on ObservableObject in existing file) |

| T-21-29 | Denial of Service | .continuous frequency on onMapCameraChange | accept | @AppStorage writes are cheap; a debounced Task can be added later if profiling shows main-thread jitter |
