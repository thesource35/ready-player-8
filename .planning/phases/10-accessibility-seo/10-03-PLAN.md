**A11Y-04: iOS accessibility labels** (per user decision: add .accessibilityLabel to all 182+ action buttons across 37 files)

For each file, find ALL `Button` views that contain `Image(systemName:)` as their label content, and ALL `Button(action:)` closures where the visible label is an icon/symbol only. Add `.accessibilityLabel("descriptive action")` directly to the Button view (NOT to a parent container).

**Pattern to apply:**
```swift
// Before:
Button(action: { deleteItem() }) {
    Image(systemName: "trash")
}

// After:
Button(action: { deleteItem() }) {
    Image(systemName: "trash")
}
.accessibilityLabel("Delete item")
```

**Also apply to NavigationLink and other tappable views with icon-only content.**

**Do NOT add .accessibilityLabel to:**
- Buttons that already have `.accessibilityLabel` (4 existing in UIHelpers.swift and SharedComponents.swift)
- Buttons whose label already contains visible Text (e.g., `Button("Save") { ... }`) -- SwiftUI auto-reads visible text
- Decorative Image(systemName:) that are NOT inside a Button
