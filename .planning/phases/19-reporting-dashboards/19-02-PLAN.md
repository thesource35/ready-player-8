4. Define a `fetchSection` helper function (per D-56 partial report pattern from RESEARCH.md Pattern 1):
   ```typescript
   const sections: Record<string, unknown> = {};
   const errors: Array<{ section: string; error: string }> = [];
   const timings: Record<string, number> = {};

   async function fetchSection(name: string, fn: () => Promise<unknown>) {
     const start = Date.now();
     try {
       const result = await Promise.race([
         fn(),
         new Promise((_, reject) => setTimeout(() => reject(new Error("Section timeout")), SECTION_TIMEOUT_MS))
       ]);
       sections[name] = { data: result, error: null, loaded_at: new Date().toISOString() };
       timings[name] = Date.now() - start;
     } catch (e) {
       errors.push({ section: name, error: String(e) });
       sections[name] = { data: null, error: String(e), loaded_at: null };
       timings[name] = Date.now() - start;
     }
   }
   ```

5. Fetch all sections in parallel via Promise.allSettled:
   - **budget**: Query cs_contracts where project relates (use project.client or join), query cs_change_orders. Pass to `computeBudgetSummary([project], contracts, changeOrders)`.
   - **schedule**: Query cs_project_tasks where project_id = id. Pass to `computeScheduleSummary(tasks)`.
   - **issues**: Query cs_rfis (filter by project context), query cs_change_orders. Pass to `computeIssuesSummary(rfis, changeOrders)`.
   - **safety**: Query cs_safety_incidents (filter by project context). Pass to `computeSafetySummary(incidents)`.
   - **team**: Query cs_project_assignments where project_id = id, query cs_team_members, query cs_activity_events where project_id = id with limit 5 ordered by created_at desc. Pass to `computeTeamSummary(assignments, members, activity)`.
   - **documents**: Query cs_document_attachments where entity_type = "project" and entity_id = id, count. Query cs_documents with gps_lat not null (photos) attached to this project, count.

6. Compute health score from sections using `computeHealthScore({ budget: sections.budget?.data, schedule: sections.schedule?.data, issues: sections.issues?.data })`.

7. Return the full ProjectReport JSON:
   ```typescript
   const duration_ms = Date.now() - startTime;
   return NextResponse.json({
     project_id: id,
     project_name: project.name,
     client_name: project.client,
     generated_at: new Date().toISOString(),
     health,
     ...sections,
     errors: errors.length > 0 ? errors : undefined,
     _meta: { duration_ms, timings },
   }, {
     headers: {
       "X-Report-Debug": JSON.stringify({ timing: duration_ms, sections: timings }),
       "Cache-Control": "private, max-age=60",
     }
   });
   ```

Import SECTION_TIMEOUT_MS from `@/lib/reports/constants`. Import all compute functions from `@/lib/reports/aggregation`. Import types from `@/lib/reports/types`.
