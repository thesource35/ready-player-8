Create the main PDF generation function:

```typescript
import jsPDF from "jspdf";
import html2canvas from "html2canvas";
import { PDF_LIGHT_THEME } from "./constants";

type PDFOptions = {
  projectName: string;
  landscape?: boolean;
  paperSize?: "letter" | "a4";
  confidential?: boolean;
  companyName?: string;
};

export async function generateReportPDF(
  reportElement: HTMLElement,
  options: PDFOptions
): Promise<Blob> { ... }
```

Implementation details:
- Auto-detect paper size (per D-34b): Check `navigator.language` -- if starts with "en-US" or "en-CA", use "letter", otherwise "a4". Allow override via options.
- Auto-detect orientation (per D-34c): `options.landscape` defaults to false for single-project (portrait), true for rollup (landscape).
- html2canvas capture: `scale: 2` (2x DPI for print, per UI-SPEC), `useCORS: true` (per Research Pitfall 1), `backgroundColor: PDF_LIGHT_THEME.background` ("#FFFFFF" per D-33).
- jsPDF constructor: `new jsPDF({ orientation: landscape ? "landscape" : "portrait", unit: "mm", format: paperSize })`.
- Margins: 19mm (0.75 inches) all sides.
- Header on each page (per D-32): 
  - Left: "ConstructionOS" text (or companyName if provided) at 10pt bold, color PDF_LIGHT_THEME.accent
  - Center: options.projectName at 14pt bold
  - Right: "Generated {Month DD, YYYY at HH:MM}" at 8pt color PDF_LIGHT_THEME.muted
- Footer on each page (per D-32):
  - Left: "Generated by ConstructionOS" at 8pt color PDF_LIGHT_THEME.muted
  - Center: "Page {N} of {total}" at 9pt
  - Right: if options.confidential, "Confidential -- {companyName}" at 8pt (per D-34f)
- Smart page breaks (per D-34j, Research Pitfall 3):
  - Capture entire reportElement as one canvas
  - Calculate image dimensions to fit page width minus margins
  - Slice at page boundaries, BUT check for CSS `data-section` attributes on child elements
  - If a section break falls within 30mm of page bottom, push to next page
- PDF naming handled by caller: `{ProjectName}-Report-{YYYY-MM-DD}.pdf` per D-31.
- Return Blob for download or preview.

Also export a helper:
```typescript
export function downloadPDF(blob: Blob, filename: string): void {
  const url = URL.createObjectURL(blob);
  const a = document.createElement("a");
  a.href = url;
  a.download = filename;
  a.click();
  URL.revokeObjectURL(url);
}

export function getPDFFilename(projectName: string): string {
  const date = new Date().toISOString().split("T")[0];
  const safeName = projectName.replace(/[^a-zA-Z0-9 ]/g, "").replace(/\s+/g, "-");
  return `${safeName}-Report-${date}.pdf`;
}
```
