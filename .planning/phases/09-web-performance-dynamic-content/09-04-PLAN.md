2. Create `supabase/migrations/001_updated_at_triggers.sql` with:
   ```sql
   -- Migration: Add updated_at auto-update triggers to all cs_* tables
   -- Run this in Supabase SQL Editor or via supabase db push
   
   -- Step 1: Create the trigger function (idempotent with CREATE OR REPLACE)
   CREATE OR REPLACE FUNCTION update_updated_at_column()
   RETURNS TRIGGER AS $$
   BEGIN
     NEW.updated_at = NOW();
     RETURN NEW;
   END;
   $$ LANGUAGE plpgsql;
   
   -- Step 2: Apply trigger to each table
   -- Using DROP IF EXISTS + CREATE to make migration idempotent
   ```

   Then for each of these tables, add:
   ```sql
   DROP TRIGGER IF EXISTS set_updated_at ON {table};
   CREATE TRIGGER set_updated_at
     BEFORE UPDATE ON {table}
     FOR EACH ROW
     EXECUTE FUNCTION update_updated_at_column();
   ```
